可能是Node.js v20.15.1版本过高，推荐降低版本
1、安装nvm-setup.exe 下载地址：https://github.com/coreybutler/nvm-windows/releases
2、验证 nvm 安装：cmd输入nvm version  输出1.2.2
3、使用指定版本node.js 
# 安装 Node.js 16.20.2 (LTS)
nvm install 16.20.2
# 在 IofTV-Screen 项目目录中使用 Node 16
nvm use 16.20.2
# 在 vue element admin 项目目录中使用 Node 20
nvm use 20.15.1
# 安装依赖并启动项目
npm install
npm start
